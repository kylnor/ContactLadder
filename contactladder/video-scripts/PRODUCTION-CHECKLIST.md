## Post-Production Workflow

For each video:

1. **Record Raw Footage**
   - [ ] Save project file
   - [ ] Export raw footage backup

2. **Edit**
   - [ ] Remove mistakes/long pauses
   - [ ] Add intro slide (5 seconds)
   - [ ] Add outro slide (5 seconds)
   - [ ] Add annotations/arrows where needed
   - [ ] Add background music (subtle, 10% volume)
   - [ ] Color correct if needed
   - [ ] Audio normalize and enhance

3. **Export**
   - [ ] Format: MP4 (H.264)
   - [ ] Resolution: 1920x1080
   - [ ] Frame rate: 30fps
   - [ ] Bitrate: 8-10 Mbps

4. **Upload to YouTube**
   - [ ] Title: "[Module Name] - howtoCloze Training"
   - [ ] Description: [Module description + link to howtoCloze.com]
   - [ ] Tags: Cloze, CRM, Real Estate, [specific topic]
   - [ ] Visibility: Unlisted
   - [ ] Copy YouTube video ID

5. **Add to Platform**
   - [ ] Log into admin: howtoCloze.com/admin
   - [ ] Edit module
   - [ ] Add video URL: youtube:[VIDEO_ID]
   - [ ] Save changes
   - [ ] Test on module page

6. **Archive**
   - [ ] Save final video file to backup drive
   - [ ] Save project file
   - [ ] Mark as complete in this checklist

## Quality Control Checklist

Before marking any video as "complete":

- [ ] Audio is clear (no background noise, consistent volume)
- [ ] Screen is easy to read (text size, resolution)
- [ ] Mouse movements are smooth (not jerky or too fast)
- [ ] All links/features demonstrated actually work
- [ ] No personal information visible (emails, phone numbers)
- [ ] Branding is consistent (intro/outro)
- [ ] Duration matches script estimate (Â±2 minutes)
- [ ] Closed captions added (YouTube auto-generate is fine)
- [ ] Module page tested and video plays correctly
