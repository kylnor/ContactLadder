## MODULE M6: OFFLINE MODE & DATA SYNC

### When You're Out of Service

**WHY (Video - 2-3 min)**

**Hook:** "Showing homes in rural areas? Underground parking? Spotty signal? Cloze mobile works offline for essential features. Why this matters: Dead zones don't mean dead productivity!"

**WHAT (Text + Visual)**

### What Works Offline

**‚úÖ Available Offline:**
- View contacts (recently accessed)
- View Agenda (cached version)
- Add notes (syncs when back online)
- View timeline (recent activity)
- Add tasks/reminders
- View cached emails

**‚ùå Requires Connection:**
- Search (fresh search)
- MAIA queries
- Send emails
- Sync new data
- Load contact you haven't viewed recently

**HOW Offline Mode Works:**

1. **Cloze caches data when online:**
   - Last 100 contacts you viewed
   - Your Agenda from last refresh
   - Recent timeline activity
2. **When offline:**
   - You can view cached data
   - Add notes/tasks (stored locally)
3. **When back online:**
   - Cloze syncs all changes automatically
   - Fresh data loads

**Best Practices for Offline Work:**

**Before going to low-signal area:**
1. **Open Cloze**
2. **View contacts you'll need** (loads them to cache)
3. **Check Agenda** (caches it)
4. **Now  work offline**

**While offline:**
- Add notes (they'll sync later)
- Create tasks
- Review cached contact info

**When back online:**
- **Pull to refresh** to sync changes
- Verify notes/tasks uploaded

## MODULE M8: MOBILE TROUBLESHOOTING

### Common Issues & Solutions

**Issue 1: App Won't Sync**

**Symptoms:** Changes not appearing, outdated data

**Solutions:**
1. **Pull to refresh** (swipe down on Agenda)
2. **Force close app** and reopen
3. **Check internet connection**
4. **Log out and log back in**
5. **Reinstall app** (last resort)

**Issue 2: Can't Find Contact**

**Symptoms:** Search returns nothing, contact should exist

**Solutions:**
1. **Check spelling**
2. **Search by email or phone** instead of name
3. **Go to People ‚Üí scroll alphabetically**
4. **Verify sync is working**

**Issue 3: Notifications Not Working**

**Symptoms:** Missing alerts for leads, Keep in Touch reminders

**Solutions:**

**iOS:**
1. **Settings ‚Üí Notifications ‚Üí Cloze**
2. **Enable "Allow Notifications"**
3. **Choose alert style**

**Android:**
1. **Settings ‚Üí Apps ‚Üí Cloze ‚Üí Notifications**
2. **Enable all notification types**

**In-App:**
1. **Cloze ‚Üí More ‚Üí Settings ‚Üí Notifications**
2. **Enable desired notification types**

**Issue 4: Battery Drain**

**Symptoms:** Cloze using significant battery

**Solutions:**
1. **Disable background refresh** if not needed
   - Settings ‚Üí General ‚Üí Background App Refresh ‚Üí Cloze ‚Üí Off
2. **Reduce sync frequency** (if option available)
3. **Update to latest app version** (bug fixes)

**Issue 5: Voice Notes Not Working**

**Symptoms:** Microphone button doesn't work

**Solutions:**
1. **Grant microphone permission:**
   - Settings ‚Üí Cloze ‚Üí Microphone ‚Üí Enable
2. **Check if microphone works in other apps**
3. **Restart phone**
