### Set Your Brand Once, Use Forever

**WHY (Video - 2-3 min)**

**WHAT (Text + Visual)**

**What Is a Marketing Email Theme?**

A Theme is your **design foundation** that controls:
- **Heading Styles:** Font, size, color for H1, H2, H3
- **Body Text:** Font, size, color, line spacing
- **Colors:** Primary, secondary, accent colors
- **Buttons:** Style, color, hover effects
- **Images:** Border radius, shadows, spacing
- **Links:** Color, underline style

**Why Use a Theme?**

1. **Consistency:** Every email matches your brand
2. **Speed:** Design decisions made once, not every time
3. **Professionalism:** Looks intentional, not random
4. **Team Alignment:** Everyone uses the same branding

**Recommended Theme Setup:**

| Element | Setting | Example |
|---------|---------|---------|
| **Heading Font** | Sans-serif, bold | Montserrat Bold |
| **Body Font** | Serif or sans-serif, readable | Georgia or Open Sans |
| **Primary Color** | Your brand color | Navy (#003366) |
| **Secondary Color** | Complementary or neutral | Gold (#FFD700) |
| **Button Style** | Solid or outline, rounded or square | Solid navy, rounded corners |
| **Link Color** | Stands out but matches brand | Gold (#FFD700) |

**HOW to Set Up Your Marketing Email Theme (Video - 4-5 min)**

*Screen recording showing:*

**Part 1: Accessing Theme Settings**

1. Go to More â†’ Settings â†’ Marketing Email
2. Tap "Theme" or "Customize Theme"
3. See theme editor

**Part 2: Setting Heading Styles**

1. **H1 (Main Headlines):**
   - Font: Montserrat (or your preferred sans-serif)
   - Size: 28px
   - Color: Navy (#003366)
   - Weight: Bold

2. **H2 (Subheadings):**
   - Font: Montserrat
   - Size: 22px
   - Color: Navy (#003366)
   - Weight: Semi-Bold

3. **H3 (Section Headers):**
   - Font: Montserrat
   - Size: 18px
   - Color: Dark Gray (#333333)
   - Weight: Bold

**Part 3: Setting Body Text**

1. **Body Text:**
   - Font: Georgia (or Open Sans for modern look)
   - Size: 16px
   - Color: Dark Gray (#333333)
   - Line Spacing: 1.6 (for readability)

**Part 4: Setting Colors**

1. **Primary Color:** Navy (#003366) - headlines, buttons
2. **Secondary Color:** Gold (#FFD700) - accents, links
3. **Background Color:** White (#FFFFFF) or light gray (#F5F5F5)

**Part 5: Setting Button Style**

1. **Button Background:** Navy (#003366)
2. **Button Text:** White (#FFFFFF)
3. **Border Radius:** 4px (slightly rounded)
4. **Hover Effect:** Darken background 10%

**Part 6: Setting Image Styles**

1. **Border Radius:** 8px (rounded corners)
2. **Shadow:** Subtle drop shadow (optional)
3. **Spacing:** 20px padding around images

**Part 7: Preview and Save**

1. Tap "Preview"
2. See sample email with your theme applied
3. Adjust if needed
4. Save Theme

**Pro Tip:**

> "Match your theme to existing brand guidelines. Use the color picker for exact hex codes. The result: Every email looks like it came from the same pro marketing team. Even if it's just you!"

**WHAT (Text + Visual)**

**What Are Building Blocks?**

Building Blocks are **reusable design elements** you create once and embed in multiple emails. Think of them as templates within templates.

**Common Building Blocks:**

| Block Type | What It Includes | Use |
|-----------|-----------------|-----|
| **Brand Header** | Logo, tagline, social icons | Top of every newsletter |
| **Brand Footer** | Contact info, website, social links | Bottom of every newsletter |
| **Legal Footer** | Unsubscribe link, address, compliance text | Required for mass email |
| **Property Card** | Property photo, price, beds/baths, CTA | Reusable listing showcase |
| **Testimonial** | Client photo, quote, attribution | Social proof |

**Why Use Building Blocks?**

1. **Consistency:** Same header/footer every time
2. **Speed:** Embed instead of rebuild
3. **Updates:** Update the block once, it updates in all emails (future sends)
4. **Team Alignment:** Everyone uses the same blocks
